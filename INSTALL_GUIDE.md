## 📋 خطوات التثبيت (3 خطوات فقط)

### الخطوة 1 — تجهيز Docker Desktop

1. حمّل وثبّت **Docker Desktop** من الرابط أعلاه
2. افتح Docker Desktop وانتظر حتى يظهر **"Docker is running"** في الشريط السفلي
3. تأكد من تفعيل **WSL2**:
   - افتح الإعدادات (Settings)
   - اختر General
   - تأكد من تفعيل "Use the WSL 2 based engine"

### الخطوة 2 — تشغيل سكريبت التثبيت

1. انقر بزر الفأرة الأيمن على ملف `SETUP_SALEHSAAS3.bat`
2. اختر **"تشغيل كمسؤول"** (Run as Administrator)
3. اتبع التعليمات على الشاشة

> **ملاحظة:** السكريبت سيقوم تلقائياً بـ:
> - فحص جميع المتطلبات
> - استنساخ المشروع في `D:\SaleHSaaS3`
> - إنشاء كلمات مرور آمنة تلقائياً
> - تحميل صور Docker (15-30 دقيقة)
> - تحميل نموذج الذكاء الاصطناعي (10-15 دقيقة)
> - تشغيل جميع الخدمات

### الخطوة 3 — فتح المنصة

بعد اكتمال التثبيت، افتح المتصفح وانتقل إلى:

```
http://localhost:8000
```

## ⚙️ إدارة المنصة

لإدارة المنصة بعد التثبيت، شغّل:

```
D:\SaleHSaaS3\scripts\windows\MANAGE_SALEHSAAS.bat
```

يوفر السكريبت خيارات:
- تشغيل / إيقاف / إعادة تشغيل الخدمات
- عرض حالة الخدمات
- عرض سجلات الأخطاء
- تحديث المنصة
- نسخ احتياطي
- تحميل نماذج ذكاء اصطناعي إضافية

## 🛠️ حل المشاكل الشائعة

### المشكلة: Docker لا يعمل
```
الحل: افتح Docker Desktop وانتظر "Docker is running"
```

### المشكلة: المنافذ مشغولة
```batch
:: تحقق من المنافذ المستخدمة
netstat -ano | findstr :8000
netstat -ano | findstr :3000
```

### المشكلة: نفاد الذاكرة
```
الحل: أغلق التطبيقات الأخرى أو زد حد الذاكرة في Docker Desktop
Settings > Resources > Memory > 12 GB
```

### المشكلة: بطء النموذج
```
الحل: تأكد من تفعيل GPU في Docker Desktop
Settings > Resources > GPU > Enable GPU
```
