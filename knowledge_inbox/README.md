# 📥 مجلد استقبال المعرفة

## كيفية الاستخدام

**ضع أي ملف هنا وسيُهضم تلقائياً في قاعدة المعرفة خلال 10 ثوانٍ!**

### أنواع الملفات المدعومة:
- 📄 PDF
- 📝 Word (.docx, .doc)
- 📊 Excel (.xlsx, .xls)
- 📋 PowerPoint (.pptx, .ppt)
- 📃 نصوص (.txt, .md, .csv)
- 📑 RTF, ODT

### ماذا يحدث بعد وضع الملف؟
1. ✅ يُكتشف الملف خلال 10 ثوانٍ
2. 🔍 يُستخرج النص عبر Apache Tika
3. ✂️ يُقسّم النص إلى أجزاء صغيرة
4. 🧠 يُولّد Embedding لكل جزء
5. 💾 يُحفظ في ChromaDB
6. 📦 يُنقل الملف إلى مجلد `knowledge_processed`

### بعد الهضم:
يمكنك السؤال عن محتوى الملف مباشرة في Open WebUI!
